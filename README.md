# CoreBluetoothBasicScanner
+ The most basic BLE scanner for iOS Devices.  
+ Scan and Print Advertising Data of BLE Devices Near Your iOS Device.
+ Written in Swift.
+ CBCentralManagerDelegate is shown in action.
